# Gosu Hangman - Keyboard Edition

A graphical implementation of Hangman using the Gosu library.

## Controls

### Gameplay

- **Letters (A-Z)**: Type any letter on your keyboard to make a guess.

### System Functions

- **[1]**: Save Game.
- **[2]**: Load Latest Game.
- **[3]**: Restart (available after game ends).

## Features

- **Direct Keyboard Input**: No selection banks or pages â€” just type to play.
- **Expanded Layout**: 1000x800 window ensures all UI elements fit comfortably.
- **Dynamic Graphics**: Stick figure draws incrementally based on incorrect guesses.
- **Premium Dark Mode**: Built with a sleek, readable color palette.

## Installation

1. Ensure Ruby is installed.
2. Install dependencies:
   ```bash
   gem install gosu
   ```

## How to Run

```bash
cd gosu-hangman
ruby main.rb
```
